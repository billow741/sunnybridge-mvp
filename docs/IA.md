## 1. 信息架构（IA）

### 模块树

```
📱 学习端 App（家长+学生共用）
├── 登录模块
│ └── 手机号+验证码登录
│ └── 手机号+密码登录（设密码后可选）
├── 课程模块
│   ├── 今日课程
│   ├── 历史课程
│   └── 课程详情
│       ├── 课堂反馈
│       └── 腾讯会议入口
├── 阅读馆模块
│   ├── 级别浏览 (L1-L6)
│   ├── 分类浏览 (绘本/短文/故事/跟读)
│   └── PDF阅读器
├── 资源库模块
│   ├── 分类浏览 (自然拼读/单词卡/推荐)
│   └── PDF预览
└── 我的模块
    ├── 孩子信息
    └── 退出登录

📱(linear) 教师端 App
├── 登录模块
│ └── 用户名+密码登录
│ └── 首次登录强制修改密码
├── 今日课程
│   └── 课程详情 + 填写反馈
└── 全部课程
    ├── 月份筛选
    └── 课程详情 + 填写反馈

🖥️ 教务后台 Web
├── 登录模块
│ └── 用户名+密码登录
├── 课程管理
├── 教师管理
├── 学生管理
├── 阅读馆管理
└── 资源库管理
```

### 页面之间的关系

```
学习端:
  登录 ──✓──→ 课程首页(默认Tab)
                 ├──→ 课程详情 → 打开腾讯会议App
                 └──→ 课程详情 → 查看反馈
  阅读馆Tab → 级别/分类筛选 → PDF阅读器(续读)
  资源库Tab → 分类筛选 → PDF预览
  我的Tab → 孩子信息 / 退出登录 → 回到登录页

教师端:
  登录 ──✓──→ 今日课程(默认Tab)
                 └──→ 课程详情 → 填写反馈 → 提交(标记completed)
  全部课程Tab → 月份筛选 → 课程详情 → 填写反馈

教务后台:
  登录 ──✓──→ 课程管理(默认页)
  侧边栏切换: 课程/教师/学生/阅读馆/资源库
  每个管理页: 列表 → 新建/编辑(弹窗)
```

## 3. 用户流程

### 3.1 学习端主流程：登录 → 上课 → 反馈 → 阅读

```
1. 打开App → 手机号+验证码登录（或手机号+密码登录）
2. 进入课程Tab → 看到今日课程列表
3. 点击课程卡片 → 进入课程详情
   3a. 上课前: 点击"进入课堂" → 跳转腾讯会议App
   3b. 下课后: 查看教师填写的反馈（课堂内容/作业/备注）
4. 切换到历史课程 → 查看往期课程和反馈
5. 切换到阅读馆Tab → 按级别(L1-L6)或分类筛选
6. 点击阅读材料 → 打开PDF阅读器
   6a. 自动续读到上次位置
   6b. 翻到最后一页 → 系统标记"已读完"
7. 切换到资源库Tab → 按分类浏览
8. 点击资源 → PDF预览
9. 切换到我的Tab → 查看孩子信息 / 退出登录
```

### 3.2 家长管理流程：查看课程 → 查看反馈 → 请假/咨询

```
1. 登录 → 进入课程Tab
2. 查看今日课程 → 了解上课时间和教师
3. 点击课程详情 → 查看反馈
   3a. 课堂内容: 本节课学了什么
   3b. 作业: 需要完成什么
   3c. 备注: 教师的额外说明
4. 查看历史课程 → 跟踪学习进度
5. 需要请假/咨询 → 联系教务（MVP阶段通过微信/电话，App内不做此功能）
6. 切换到阅读馆 → 推荐孩子阅读
7. 查看孩子的阅读进度（已读/未读标记）
```

### 3.3 教师流程：查看课表 → 上课 → 填写反馈 → 推荐资源

```
1. 打开App → 用户名+密码登录
2. 进入今日课程Tab → 查看今日课表
3. 点击课程卡片 → 进入课程详情
 3a. 查看学生信息
 3b. 点击"进入课堂" → 跳转腾讯会议
4. 下课后 → 在课程详情页填写反馈
   4a. 课堂内容（必填）
   4b. 作业（选填）
   4c. 备注（选填）
   4d. 点击"提交反馈" → 系统自动标记课程为 completed
5. 切换到全部课程Tab → 月份筛选查看历史课程
6. 已提交反馈的课程显示"已完成"状态
7. 未提交反馈的课程显示"待反馈"提示
```
